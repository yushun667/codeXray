## 阅读顺序建议

1. **系统总览**：先读 [04-系统设计/系统框架与数据流.md](04-系统设计/系统框架与数据流.md)。
2. **扩展集成**：再读 [00-主仓库/扩展架构与集成.md](00-主仓库/扩展架构与集成.md) 与 [00-主仓库/命令与视图清单.md](00-主仓库/命令与视图清单.md)。
3. **各子模块**：按需查阅 01 / 02 / 03 下对应文档。
4. **UI 与交互**：实现或验收 UI 时参考 [02-可视化界面/UI与交互逻辑.md](02-可视化界面/UI与交互逻辑.md)。
